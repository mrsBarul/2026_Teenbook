## Схема связей между темами

В рамках темы **«Мои отношения в школе»** были выделены основные понятия, связанные с взаимодействием учеников и учителей в школьной среде.

Ключевые сущности предметной области:

- средняя школа  
- ученик  
- учитель  
- общение  
- дружба  
- конфликт  
- травля  

Основная логика связей между понятиями:

- **Средняя школа** является средой, в которой происходит взаимодействие учеников и учителей.
- **Учитель** взаимодействует с **учениками** в процессе обучения.
- **Ученики** участвуют в **общении** друг с другом.
- В результате общения могут возникать **дружеские отношения**.
- Иногда общение может приводить к **конфликтам**.
- В некоторых случаях конфликт может перерасти в **травлю (буллинг)**.

Таким образом, онтология описывает социальные взаимодействия внутри школьной среды.

## Примеры SPARQL-запросов

Для извлечения знаний из базы Wikidata использовались SPARQL-запросы.

Пример запроса для получения связей между выбранными сущностями:

```sparql
SELECT ?source ?sourceLabel ?property ?propertyLabel ?target ?targetLabel WHERE {
  VALUES ?source {
    wd:Q37226
    wd:Q48282
    wd:Q159334
    wd:Q491
    wd:Q240126
    wd:Q180684
    wd:Q11024
  }

  VALUES ?target {
    wd:Q37226
    wd:Q48282
    wd:Q159334
    wd:Q491
    wd:Q240126
    wd:Q180684
    wd:Q11024
  }

  ?source ?directProp ?target .
  ?property wikibase:directClaim ?directProp .

  FILTER(?source != ?target)

  SERVICE wikibase:label {
    bd:serviceParam wikibase:language "ru,en" .
  }
}
```

Данный запрос позволяет найти прямые связи между выбранными сущностями в базе знаний Wikidata.

Результат выполнения запроса был сохранён в файле: [data/wikidata_export_contact.json](data\wikidata_export_contact.jпшson)
