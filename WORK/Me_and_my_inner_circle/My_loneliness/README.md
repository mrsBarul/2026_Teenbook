## Основные понятия из онтологии

- **одиночество** — внутреннее состояние человека в отсутствие близких;
- **социальная изоляция** — отсутствие возможности присоединиться к обществу;
- **социальная связь** — противоположность одиночеству, ощущение близости и связи с другими людьми;
- **дружба** — один из способов ослабить одиночество;
- **привязанность** — отношения и эмоции, которые связывают людей;
- **отчуждение** — состояние, близкое к одиночеству, часто усиливает стресс;
- **дистресс** — стресс, связанный с одиночеством и тревогой;
- **эмпатия** — способность понять чувства другого человека;
- **поддержка** — действия, которые помогают человеку не оставаться наедине со своими проблемами;
- **самооценка** — оценка себя, которая может падать при одиночестве.


## Как связаны сущности в WikiData

Данные связи взяты из `wikidata_export_contact.json`, который формируется скриптом `contact.py`.

## SPARQL-запросы

В папке `scripts` лежат два основных скрипта:

- [`query.py`](/WORK/Me%20and%20my%20inner%20circle/My%20loneliness/scripts/query.py) — получает названия и описания понятий (если есть русское описание).
- [`contact.py`](/WORK/Me%20and%20my%20inner%20circle/My%20loneliness/scripts/contact.py) — получает из Wikidata связи между понятиями (тройки `source-property-target`).
